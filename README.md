---

RigRadar searches eBay in real time and tells you if a GPU or CPU listing is a good deal or overpriced. I built this because I kept overpaying for parts and wanted market data

---

## What it does

- Pulls live eBay listings for any GPU or CPU you search
- Shows average, lowest, and highest prices
- Labels each listing as Good Deal or Overpriced
- Tracks your recent searches and suggests parts as you type
- Compare two parts side by side
- Built-in profit calculator if you're flipping parts
- Filter and sort results by price

---

## Stack

Python, Flask, HTML, CSS, JavaScript, eBay Browse API — deployed on Render.
